# Newport Maeve Chronicles - Brand Website

A sophisticated dark fantasy brand website for the Newport Maeve Chronicles prequel, featuring Miroslav Dion and the world of shadows and power.

## Project Structure

```
newport-maeve/
├── index.html          # Main website file
├── README.md           # This file
├── favicon.ico         # Website favicon (add your own)
└── images/             # Image directory (create this folder)
    ├── hero-bg.jpg     # Hero section background
    ├── miroslav-dion.jpg
    ├── map.jpg
    └── og-image.jpg    # Open Graph image for social sharing
```

## Image Placeholders

The website includes placeholders for the following images. Upload these files to deploy the site:

### Required Images

1. **hero-bg.jpg** - Hero section background image (recommended: 1920x1080px or larger)
2. **miroslav-dion.jpg** - Main character portrait (recommended: 600x800px)
3. **map.jpg** - Newport Maeve world map (recommended: 1600x900px)
4. **og-image.jpg** - Open Graph image for social media sharing (recommended: 1200x630px)

## Customization

### Colors
The website uses a dark fantasy color scheme:
- Primary Gold: `#d4af37`
- Light Gold: `#ffd700`
- Dark Background: `#0a0a0a`
- Card Background: `rgba(20, 20, 40, 0.8)`

To customize, edit the CSS color values in the `<style>` section.

### Fonts
- Display Font: Cinzel (serif)
- Body Font: Lora (serif)

These are loaded from Google Fonts. To change, update the `<link>` tag in the `<head>`.

### Content
All text content is in the HTML file. Simply edit the text within the HTML tags to update content.

## Deployment

### Local Testing
Open `index.html` in a web browser to test locally.

### Web Hosting
Upload all files to your web host:
1. `index.html`
2. `favicon.ico`
3. All image files in an `images/` folder
4. Character sheet PDF/image files

### Manus Deployment
If deploying on Manus:
1. Upload the HTML file
2. Upload images to the static assets folder
3. Update image paths in the HTML if necessary

## Performance Tips

1. Optimize images before uploading (compress JPG/PNG files)
2. Use WebP format for better performance (with JPG fallbacks)
3. Lazy load images if adding more than 5 images
4. Monitor Core Web Vitals

## Analytics

Add your analytics tracking code (Google Analytics, etc.) in the `<head>` section before the closing `</head>` tag.
